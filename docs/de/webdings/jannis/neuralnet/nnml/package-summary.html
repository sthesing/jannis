Package de.webdings.jannis.neuralnet.nnml

Class Summary

BiNetToNNML BiNetToNNML is used to convert a neural net of BiNeurons to a String containing a NNML representation of that net.
NetToNNML NetToNNML is used to convert a neural net of to a String containing a NNML representation of that net.
NNMLToBiNet NNMLToBiNet is used to construct a neural net with BiNeurons from a NNML representation.
NNMLToNet NNMLToNet is used to construct a neural net from a NNML representation.
